# Filter Odd Numbers from Array

You will be given array of numbers and you need to remove the elements that are odd.<br>

Let's begin by giving an example.

```
Given: [1, 2, 3, 4]
Output: [2, 4]

Given: [-1, 100, -3]
Output: [100]

```
